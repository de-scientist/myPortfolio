# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

## Navigate into the Project

```bash
cd portfolio
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

Visit:

```text
http://localhost:5173
```

---

# 📦 Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
